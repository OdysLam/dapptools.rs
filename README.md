# Dapptools.rs

The landing page for the fastest dapptools out there.

It's made with Gatsby and Tailwind.

## Contributing

It's a landing page. Sure, why not.

### Install Gatsby

Follow [Gatsby docs](https://www.gatsbyjs.com/docs/tutorial/part-0/).

### Run locally

```bash
git clone https://github.com/odyslam/dapptools.rs
cd dapptools.rs
gatsby develop
```
Don't forget to run `yarn format` before making a PR.
